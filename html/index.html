Boxes and Bars

Boxes are the most versatile component of the Start library. These can function as navigation menus, toolboxes, or any generic container.

Navigation Boxes

Boxes for horizontal navigation should be filled with unordered lists (``ul``)

</nav>

Browser Support

Which browsers does start.css support? Latest.

IE11+, Chrome, Firefox 16+, and Safari (sort of) are all auto updating browsers. Because of this shift in the browser support ecosystem, we state the current latest browsers on the day of a Start.css stable release are the browsers that version supports.

For older browser support, download an earlier version of Start.css.

End Concatenation

This is a small trick to prevent the last line of a paragraph from containing only one word.

So instead of:

lorem ipsum dolor sit
amet

End concatenation produces:

lorem ipsum dolor
sit amet

To use, simply add the class .s-endconcat or use the jQuery plugin

$(element).s_endconcat()

Where element is the element to have enconcatenation applied.

Grid

Start uses a 12 column grid. To create a gridded row, start with an .s-row container.

<section class="s-row"></section>

Columns are defined with the .s-col class and with .s-x measuring classes where "x" is a number between 1-12.

Using .s-col without a measuring class will default to .s-12

s-col s-4
s-col s-8

<section class="s-row">
  <div class="s-col s-4">s-col s-4</div>
  <div class="s-col s-8">s-col s-8</div>
</section>

Responsive Grid

The grid is designed to operate mobile-first, so there is no class to target devices with widths less than 320px.

To specify a column size for a larger screen, use the additional size prefixes sm, md, lg, and xl.

(s-col s-md-4) 4 columns in medium width
(s-col s-md-8) 8 columns in medium width

(s-col) Mobile devices tend to use one-column layouts more often.
(s-col) So why bother specifying every time?
(s-col s-lg-6) Write only the classes needed for larger screens.
(s-col s-lg-6) Keep your HTML tidy.
(s-col s-lg-4) Responsive design
(s-col s-lg-4) is
(s-col s-lg-4) easy.

Container

To contain the page in a nice, centered wrapper, use .s-wrapper

By default, this will have no effect for pages under 1200px. Screens larger than this will keep your site in a 1200px wrapper with equal spacing on the left and right.

For different sizes, add an additional class with the form s-[size] where [size] is xs, sm, md, or lg.

<div class="s-wrapper s-lg"></div>

Growls

Growls are unobtrusive pop up messages called via JavaScript. A basic call might look like this:

$.sGrowl({
   icon: 'save',
   title: 'Saved',
   text: 'Your document has been saved'  
 })

Parameters

Parameter Default Description
icon — Icon to appear before the heading. Choose from the Font Awesome 4 library without the preceding "fa-"
title — Bold-faced title to appear on the first line.
text — Main text under the title. HTML allowed.
delay (Infinite) Delay in miliseconds before dismissing growl. Ignore this parameter to keep growl permanent.
container #s-growls HTML element where growl should be sent. Write in CSS notation.
color green Color of growl (red/orange/yellow/green/blue/violet).
class — Additional classes to apply to growl. Separate multiple classes with a space.

Advanced

Here is a growl that will appear in red in the container below and disappear after 3 seconds.

Growl will appear above.

$.sGrowl({
  color: 'red', 
  icon: 'book',
  title: 'Test',
  text: 'Testing...1, 2, 3', 
  delay: 3000, 
  container: '#example-container'
})

Progress

Use the default HTML5 progress element with the .s-box class.

<progress class="s-box s-3" value="30" max="100"></progress>

Add .s-[color] for alternate colors.

<progress class="s-box s-red" value="30" max="100"></progress>

Anchor Adjust

This will create a little margin trick allowing for headings to adjust vertically when retrieved by an anchor link.

Try out the page-level links on this page as an example.

With the basic library, this approximates some space below a fixed navbar. If you need more customization, try the semantic version of Start.css

<h2 class="s-anchorAdjust" id="link">Title</h2>
<a href="#link">Go to title</a>
